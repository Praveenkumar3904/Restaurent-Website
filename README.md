# 🍽️ Restaurant Menu & Reservation Website

A modern, responsive restaurant website that allows users to explore menu items and book tables online.

---

## 🚀 Live Demo

👉 https://praveenkumar3904.github.io/Restaurent-Website/

---

## 📌 Features

* 🍴 Attractive restaurant-style UI
* 📋 Categorized menu (Starters, Main Course, Desserts, Beverages)
* 🔍 Menu filtering using JavaScript
* 📅 Table reservation form with validation
* 📱 Fully responsive design
* 🎨 Smooth scrolling and animations

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## ⚙️ Run Locally

```bash
git clone https://github.com/praveenkumar3904/Restaurent-Website.git
```

Open `index.html` in your browser.
